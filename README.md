## 基本介绍

`pwn` （据说）是黑客俚语，意指通过计算机漏洞获得对计算机系统的控制权，从而可以对计算机系统进行各种破坏、修改、控制。

在本教程中，你将会学到：

- 汇编语言基础
- 逆向分析
- gdb调试工具的使用
